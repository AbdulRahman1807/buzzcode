# ⚡ BuzzCode — Coding Contest Buzzer

A real-time buzzer system for coding contests, quizzes, and classroom competitions.

Built using Firebase Realtime Database for instant synchronization.

---

## 🚀 Features

- ⚡ Real-time buzzer (first tap wins)
- 👨‍🏫 Host-controlled questions
- 👥 Live participant list
- 🧠 Score tracking
- ⏱️ Configurable buzz delay
- 📱 Works on mobile browsers
- 🔄 Auto reconnect & session restore

---

## 🛠️ Tech Stack

- Vanilla HTML, CSS, JavaScript
- Firebase Realtime Database
- GitHub Pages (hosting)

## ▶️ How to Use

### Host
1. Open the app
2. Click I'm the Host
3. Enter:
   - Room code
   - Questions (one per line)
   - Buzz delay
4. Start session

### Participants
1. Open the app
2. Click I'm a Participant
3. Enter:
   - Room code
   - Name
4. Wait for question → buzz!

---

## 🔧 Firebase Setup

1. Go to https://console.firebase.google.com  
2. Create project  
3. Enable Realtime Database (test mode)  
4. Copy firebaseConfig  
5. Paste into app on first launch  

---

## ⚠️ Notes

- This is a client-only app (no backend)
- Firebase handles all real-time sync
- Best used on a shared WiFi network
