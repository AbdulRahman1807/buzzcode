⚡ Firebase Setup Required

This app needs a Firebase project for real-time sync. One-time setup, ~3 minutes.

Step 1 — Create a Firebase project
Go to console.firebase.google.com → "Add project" → any name → disable Analytics → Create.
Step 2 — Enable Realtime Database
Build → Realtime Database → Create Database → Start in test mode → Enable.
Step 3 — Get your config
Project Settings (⚙) → General → "Your apps" → click </> Web → Register app → copy the firebaseConfig object → paste below.
